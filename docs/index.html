Debug Go programs

A desktop debugger for Go powered by Delve. Multi-editor config support, theme engine, system tray, and auto-updates. Built with Wails + Svelte.

Download Quick Start

Everything you need to debug Go

🔍

Full DAP Client

Breakpoints, stepping, call stack, variables, evaluate expressions. All powered by the Debug Adapter Protocol over Delve.

📂

Multi-Editor Import

Auto-detects debug configs from VS Code, Zed, and GoLand. Scans your system and imports with one click.

🎨

Theme Engine

8 bundled themes including One Dark, GitHub, Catppuccin, Dracula, Nord. Live preview, install custom themes from file.

🖥️

System Tray

Manage sessions from the menu bar. Left-click popup with controls, right-click for quick config switching.

⌨️

Keyboard First

Command palette with fuzzy search, full keyboard navigation in settings, F-key debug shortcuts. Vim mode optional.

🚀

Multiple Sessions

Run several debug targets at once. Session tabs in the title bar, each with independent terminal output and variables.

📡

Port Conflict Detection

Detects "address already in use" errors and offers to kill the blocking process, then auto-retries the debug session.

🔄

Auto-Updates

Checks GitHub Releases on launch and notifies you when a new version is available. One-click to view the release.

Get started in 30 seconds

Requires Go 1.21+, Delve, Node.js 18+, and Wails v3 CLI.

# Install Delve
go install github.com/go-delve/delve/cmd/dlv@latest

# Install Wails v3
go install github.com/wailsapp/wails/v3/cmd/wails3@latest

# Clone and run
git clone https://github.com/Patrick-web/DelveUI.git
cd DelveUI
wails3 dev

Or download a pre-built binary from GitHub Releases.
